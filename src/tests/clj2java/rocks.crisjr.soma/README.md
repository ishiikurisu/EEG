# rocks.crisjr.soma

A stupid app to sum numbers from a YAML list.

## Installation

Download from nowhere. Really.

## Usage

Give the YAML file name and our app will do the work for you.

    $ java -jar rocks.crisjr.soma-0.1.0-standalone.jar file.yml
