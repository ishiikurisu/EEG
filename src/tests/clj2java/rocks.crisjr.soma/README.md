# rocks.crisjr.soma

A stupid app to sum numbers from a YAML list.

## Installation

Download from nowhere. Really.

## Usage

Pipe the YAML file into our app and it will do the work for you.

    $ more file.yml | java -jar rocks.crisjr.soma-0.1.0-standalone.jar
